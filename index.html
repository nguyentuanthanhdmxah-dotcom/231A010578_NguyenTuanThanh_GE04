Mô tả dự án - Movie Discovery App

Đây là dự án “Movie Discovery App” được làm theo đề thi kết thúc học phần Lập trình Web.

  • Bài 1: Giao diện sử dụng CSS Grid, thẻ phim có hover effect, hỗ trợ Light/Dark Mode.
  • Bài 2: Tương tác dữ liệu bằng JS, lọc phim theo thể loại + tìm kiếm, debounce tối ưu hiệu năng.
  • Bài 3: Modal hiển thị chi tiết phim, hiển thị poster, mô tả, đạo diễn, thể loại. Link tới bài tập 34.
  • Bài 4: Tối ưu hình ảnh và triển khai website trên Netlify/Vercel.
